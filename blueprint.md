# Advanced Care Website Blueprint

## Overview

This document outlines the design, features, and structure of the Advanced Care hospital website. The site is a modern, responsive, and professional platform designed to provide information about the hospital, its services, and its medical team. The website is built using HTML, CSS, and JavaScript, and it is designed to be accessible and user-friendly.

## Design and Styling

*   **Color Palette:** The website uses a professional and trustworthy color palette, with a primary color of `#1A237E`, a secondary color of `#00BCD4`, and an accent color of `#E3F2FD`. The background is a clean light grey (`#F5F5F5`), and the text is a standard dark grey (`#333`).
*   **Typography:** The website uses the 'Roboto' font, which is a clean and readable sans-serif font.
*   **Layout:** The website has a classic and professional layout, with a focus on providing information in a clear and concise manner. The homepage features a hero section, a "Commitment to You" section, a "Why Choose Us" section, a statistics section, and a "Meet Our Doctors" section. The layout is now fully responsive and all alignment issues have been fixed.

## Features

*   **Homepage:** The homepage provides a comprehensive overview of the hospital. It includes a hero section with a clear call to action, a section highlighting the hospital's commitment to patients, a "Why Choose Us" section, a statistics section (now in a vertical format), and a section introducing some of the hospital's doctors.
*   **About Us Page:** The "About Us" page provides a comprehensive look at the hospital's history, mission, and values.
*   **Services Page:** The "Services" page provides a detailed overview of the hospital's medical services.
*   **Doctors Page:** The "Doctors" page introduces the hospital's medical team of 15 doctors. Each doctor has a card that, when clicked, opens a modal with more information and a booking form.
*   **Appointment Page:** The "Appointment" page allows users to book an appointment with a doctor. The form is fully restored and includes a dropdown with all 15 doctors.
*   **Blog Page:** The "Blog" page is a valuable resource for patients, with articles on a variety of health topics.
*   **Contact Page:** The "Contact" page provides various ways for users to get in touch with the hospital.
